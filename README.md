#  Lost Found App

¿Cansado de perder tus pertenencias y gastar dinero en reponerlas?  
Con **Lost Found**, eso se acabó. Nuestra aplicación está diseñada para ayudarte a **encontrar lo que se pierde** y **reconectar personas con sus objetos** de forma rápida, sencilla y segura.

---

##  Nuestra misión

Crear una comunidad colaborativa donde cada usuario pueda reportar, encontrar y devolver objetos perdidos.  
Porque creemos que **ayudar también puede ser parte de la tecnología.**

---

##  Funcionalidades principales

-  **Publicar objetos perdidos**  
  Crea un post indicando qué se perdió, dónde fue y adjunta una foto para facilitar su identificación.

-  **Reportar objetos encontrados**  
  Informa a la comunidad cuando encuentres algo y ayuda a que regrese con su dueño.

-  **Contacto en tiempo real**  
  Chatea directamente con la persona que perdió o encontró un objeto, de forma segura y privada.

-  **Perfil de usuario**  
  Visualiza tus publicaciones, datos personales y tu historial dentro de la comunidad.

-  **Sistema de cuentas seguro**  
  Registro, inicio y cierre de sesión con autenticación robusta mediante Firebase.

---

##  Conclusión

**Lost Found** no es solo una app — es una red solidaria.  
Cada reporte, cada mensaje y cada reencuentro hace del mundo un lugar un poco mejor.  
Si alguna vez pierdes algo, **estaremos aquí para ayudarte a encontrarlo.**
